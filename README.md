## IOS

```sh
pod install

```

pod install 안되면
ruby 버전화긴하기. 3.2.2 여야함
아니면 변경 하고 업뎃

```
rbenv versions

echo 'eval "$(rbenv init -)"' >> ~/.zshrc
source ~/.zshrc

```
